## 前言

我的博客基于angular，使用了devui组建。

## 环境

- 一台服务器
- 一个用来写代码的设备

## 具体搭建过程

### 服务器准备

**在服务器上**

新建工作目录和代码仓库
> mkdir blog
>
> mkdir blog.git
>
> cd blog.git
>
> git init --bare

修改hooks中的一个文件，使得`git push`完成之后，自动帮我们做一些事情。

现在是在`blog.git`文件夹下

> cd hooks
>
> vim post-update
> 
> //写脚本 
> 
> sudo chmod +700 post-update

脚本如下

```sh
#!/bin/bash
cd /root/blog
unset GIT_DIR
git pull
cd /root/blog/src/allArticleResources
python3 make_json.py
```

脚本的作用：进入工作目录`/root/blog` ，在这个目录里我们把代码仓库同步更新下来。进入文章管理目录，运行更新list.json的python文件。这样，每次更新文章的时候都会自动重新生成文章列表。
